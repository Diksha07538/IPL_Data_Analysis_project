IPL Data Analysis

This project performs Exploratory Data Analysis (EDA) on the Indian Premier League (IPL) dataset using Python. The goal is to extract meaningful insights about teams, players, batting, bowling, and match performances through data analysis and visualization.

- Project Objectives
Clean and preprocess IPL datasets
Perform Exploratory Data Analysis (EDA)
Analyze team performances across seasons
Identify top batsmen and bowlers
Compare winning trends and toss decisions
Visualize important statistics using charts and graphs
Derive meaningful insights from IPL historical data
- Technologies Used
Python
Jupyter Notebook
Pandas
NumPy
Matplotlib
Seaborn
- Key Analysis Performed
Top run scorers
Highest wicket takers
Team-wise wins
Toss winner vs Match winner analysis
Venue analysis
Most Player of the Match awards
Batting and bowling performance
Season-wise trends
Data visualization using various charts
